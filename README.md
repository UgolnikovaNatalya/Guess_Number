# Guess_Number
Реализация игры с помощью LiveData &amp; MVVM
Данная игра позволяет играть с Android или с другом.
Правила игры:
Друг или Android загадывают число от 1 до 100. У вас есть 7 попыток отгадать число, которое было загадано.
